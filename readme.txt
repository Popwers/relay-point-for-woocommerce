== Description ==

Relay point for WooCommerce adds additional shipping methods to manually create relay points.
But it's not just about creating a new shipping method, this plugin displays a pick list instead of displaying relay points one by one!

= Note for developers =

You can participate to the improvement of this plugin by going on the GitHub repo: https://github.com/Popwers/woocommerce-relay-shipping

== Frequently Asked Questions ==

= How to use this plugin ? =

Just install it and go on your shipping methods to add some additional relay point, then they will display on the checkout screen !
