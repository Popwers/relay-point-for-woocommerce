Create manually relay point on Woocommerce and show them in a better way selection !

== Description ==

Relay point for Woocommerce add extra shipping methods for create manually relay point.
But isnt about create just a new shipping methods, this plugin display a list of selection instead of display the relay point one by one !

= Note for developers =

You can participate in the improvement of this plugin by going to the GitHub repo : https://github.com/Popwers/woocommerce-relay-shipping

== Frequently Asked Questions ==

= How to use this plugin ? =

Just install it and go on your shipping methods to add some additional relay point, then they will display on the checkout screen !
